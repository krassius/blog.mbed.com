    about..<br>

  <ul>

    </li>
    <li>@toyowata ( <a href=
    "http://mbed.org/users/MACRUM/">http://mbed.org/users/MACRUM/</a>

    environment
    </li>
    <li>Junichi KATSU (@jksoft913 <a href=
    "http://mbed.org/users/jksoft913/">http://mbed.org/users/jksoft913/</a>
    ) : mbed robot base board
    </li>
    <li>@tosihisa (<a href=
    "http://mbed.org/users/tosihisa/">http://mbed.org/users/tosihisa/</a>
    ) : Custom GPS logger using mbed
    </li>
    <li>@ytsuboi ( <a href=
    "http://mbed.org/users/ytsuboi/">http://mbed.org/users/ytsuboi/</a>
    ) : Pati-bed (Compatible mbed) board
    </li>
    <li>@yosimo : RoboCup with mbed
    </li>
    <li>Hiroshi Suga (@okini3939 <a href=
    "http://mbed.org/users/okini3939/">http://mbed.org/users/okini3939/</a>)
    </li>
